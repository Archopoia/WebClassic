# ClassicWeb

*The Order* — a treatise on **classical web design**: a webpage built as a classical building, governed by order, proportion, hierarchy, rhythm, and a single module, after the manner of Giacomo Barozzi da Vignola.

## Contents

- **`index.html`** — the finished page, a self-contained demonstration where the screen is the site, the layout is the plan, typography is the ornament, navigation is the circulation, and content is the inhabitant.
- **`Classical_Web_Design`** — the design method: establishing the module, proportion, hierarchy, and rhythm.
- **`Classical_Vignola`** — notes drawn from Vignola's principles of the classical orders.

## Viewing

Open `index.html` in any modern browser. No build step or dependencies are required.
